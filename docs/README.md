# Documentação

Aqui estão os documentos técnicos do projeto. Cada um cobre um aspecto específico da arquitetura.

## O que tem aqui

**DIAGRAMA_COMPONENTES.md** - Mostra como tudo se conecta. Arquitetura AWS, componentes da aplicação, Kubernetes, monitoramento... tudo visualizado.

**DIAGRAMA_SEQUENCIA.md** - Fluxos das principais operações. Desde a autenticação até a entrega de uma ordem de serviço. Cada passo detalhe por detalhe.

**RFC.md** - Decisões técnicas que tomei. Por que AWS, por que PostgreSQL, por que JWT... análise das alternativas e o que escolhi.

**ADR.md** - Decisões arquiteturais. Arquitetura Hexagonal, máquina de estados, como fazer as coisas certas no código.

**BANCO_DE_DADOS.md** - Tudo sobre o banco de dados. Diagrama ER, schema SQL, queries úteis, relacionamentos explicados.
